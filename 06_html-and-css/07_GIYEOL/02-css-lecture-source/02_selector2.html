선택자[속성=속성값] {설정 내용}의 형식으로 사용됨

후손 선택자와 자손 선택자

            후손 선택자: 선택자 후손 선택자{설정 내용}의 형시으로 사용

동위 선택자

동위 관계(형제 관계)에서 뒤에 위치한 태그를 선택할 때 사용함

바로 뒤의 요소를 선택 시: 선택자 a + 선택자 b{설정 내용}으로 사용함,
            뒤에 있는 모두를 선택할 때는 선택자 a ~ 선택자 b{설정 내용}으로 사용됨

반응 선택자

사용자의 움직임에 따라 달라지는 선택자

사용자가 클릭한 태그: 선택자: active{설정 내용},
            사용자의 마우스가 올라가 있는 태그: 선택자:hover{설정 내용}으로 사용됨
active 테스트
hover 테스트

상태 선택자

입력 양식의 상태에 따라 선택되는 선택자

체크 상태의 input 태그 선택 : input 태그 :checked{설정내용}

초점이 맞춰진 input태그 선택



사용가능한 input태그 선택

태그:enabled{설정 내용}, 태그 :disabled{설정 내용}
